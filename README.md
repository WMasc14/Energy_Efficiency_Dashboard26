# ⚡ Dashboard de Eficiência Energética para Fábrica

## 📊 Descrição do Projeto

Este projeto consiste no desenvolvimento de um **dashboard interativo de eficiência energética** para uma fábrica, criado com **Python** e **Streamlit**.

O objetivo é monitorizar, analisar e visualizar o consumo energético da fábrica, permitindo identificar padrões, desperdícios e oportunidades de otimização.

---

## 🎯 Objetivos

* Monitorizar o consumo de energia em tempo real ou histórico
* Identificar picos de consumo e períodos de ineficiência
* Comparar consumo entre setores/máquinas
* Apoiar a tomada de decisões para redução de custos energéticos
* Promover sustentabilidade e eficiência operacional

---

## 🛠️ Tecnologias Utilizadas

* Python 🐍
* Streamlit 🚀
* Pandas 📊
* NumPy 🔢
* Matplotlib 📈
* Seaborn 🎨 (opcional para visualizações)

---

## 📊 Funcionalidades do Dashboard

O dashboard inclui:

### 🔹 Visão Geral

* Consumo total de energia
* Consumo médio diário/mensal
* Indicadores de eficiência energética (KPIs)

### 🔹 Análise Temporal

* Evolução do consumo ao longo do tempo
* Identificação de picos de consumo
* Comparação entre períodos

### 🔹 Análise por Setor/Máquina

* Consumo energético por área da fábrica
* Ranking dos maiores consumidores
* Identificação de ineficiências

### 🔹 Visualizações Interativas

* Gráficos dinâmicos com filtros
* Seleção de períodos personalizados
* Comparação entre variáveis
